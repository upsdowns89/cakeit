# 🎂 [Project] 전국 커스텀 케이크 통합 플랫폼 - 가게 상세 및 관리 시스템

본 문서는 구매자용 '가게 상세 페이지'와 판매자용 '내 가게 관리' 시스템의 UI/UX 및 데이터 로직을 정의합니다.

---

## 1. UI 전체 레이아웃 (Layout Architecture)
- **구조**: Sticky GNB(상단) + 스크롤 본문 영역 (Thumbnail -> Menu Tabs -> Tab Content).
- **인터랙션**: 구매자 페이지 하단에 '주문하기' 플로팅 버튼 상시 노출.

---

## 2. 컴포넌트별 상세 기획

### [2-1] 상단 GNB & 대표 썸네일
- **GNB (Detail 타입)**
    - 왼쪽: 뒤로가기, 홈 버튼
    - 오른쪽: 북마크, 공유하기 (Device Native Share)
- **대표 썸네일**
    - 구성: N개의 이미지 스와이프 (4:3 비율 라운드 카드)
    - 인디케이터: 안쪽 하단 위치
    - 셀러 관리: 사진 추가/삭제, 노출 순서 편집 기능

### [2-2] 메뉴 탭 상세 (Menu Tabs)

#### 1. 홈 (Summary)
- **가게 정보**: 간략한 소개글 노출
- **스케줄**: 드롭다운 입력 필드 (가게 타임테이블 조회 기반 가장 빠른 날짜 자동 선택)
- **섹션 프리뷰**: 추가정보, 메뉴, 사진, 리뷰 섹션의 요약본 노출 (클릭 시 해당 탭 이동)

#### 2. 공지 (Notice)
- **기능**: 셀러가 포스팅한 공지사항 리스트
- **특징**: 외부 플랫폼(인스타그램 등) 글 가져오기 기능 포함 예정

#### 3. 메뉴 (Pricing Engine)
- **UI**: 상단 [사이즈 탭] + 하단 [디자인 타입 리스트]
- **데이터 로직**: 사이즈(도시락, 1호 등) 선택 시 해당 사이즈의 디자인 타입별 구간 가격(N만~N만) 노출
- **셀러 관리**: 사이즈별 기본가 및 디자인 타입별 추가 비용 설정

#### 4. 사진 (Gallery)
- **기능**: 리뷰에서 업로드된 사진만 모아보는 갤러리 (3열 그리드)
- **인터랙션**: 사진 클릭 시 전체화면 상세 보기 (스와이프 가능)
- **연동**: 하단 '리뷰보기' 플로팅 버튼을 통해 리뷰 상세로 이동

#### 5. 리뷰 (Review)
- **필터**: '사진 포함 리뷰만 보기' 체크박스 기능
- **외부 연동**: 셀러가 설정한 네이버/카카오/구글 리뷰 링크로 이동하는 버튼 제공

#### 6. 정보 (Info & Location)
- **픽업 안내**: 제목별 아코디언(드롭다운) UI. 사진(N개)과 설명 포함
- **영업 시간**: 셀러가 설정한 요일별 영업시간 노출
- **위치**: 네이버 지도 API 연동 (기본값: https://naver.me/5CFP6Vhb)

---

## 3. [Seller] '내 가게 관리' 스펙 (Admin)

### A. 가게 프로필 설정
- 대표 썸네일 업로드 및 순서 변경
- 가게 소개글 및 공지사항 작성/수정

### B. 상품 및 가격 관리 (Menu Builder)
- 케이크 사이즈별 탭 생성/삭제
- 사이즈 내 디자인 타입(레터링, 3D 등)별 가격 범위 설정

### C. 픽업 및 영업 가이드 설정
- 픽업 안내 아코디언 항목 추가 (제목, 사진, 본문 입력)
- 요일별 상세 영업시간 및 임시 휴무 설정

### D. 외부 링크 및 지도 설정
- 네이버/카카오/구글 리뷰 URL 등록
- 지도에 노출될 가게 주소 검색 및 좌표 설정
