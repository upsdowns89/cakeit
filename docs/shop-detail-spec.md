# 샵 상세 페이지 역기획서

> **파일**: `src/app/shop/[id]/page.tsx` (바이어), `src/app/seller/shop/page.tsx` (셀러 내 스토어)
> **원칙**: 셀러 "내 스토어"의 보기 모드 UI는 바이어 "샵 상세"와 **1:1 완전 동일**해야 함. 편집 버튼/폼만 셀러 전용.

---

## 1. 전체 레이아웃

```
┌─────────────────────────────────┐
│  GNB (상단 네비게이션)            │
├─────────────────────────────────┤
│  Hero 이미지 캐러셀              │
├─────────────────────────────────┤
│  가게 기본 정보                   │
│  (영업 상태 · 별점 · 이름 · 주소 · 설명)│
├─────────────────────────────────┤
│  Tab Bar                         │
│  [메뉴(기본)] [공지] [사진] [리뷰] [정보] │
├─────────────────────────────────┤
│  Tab Content (선택된 탭 내용)      │
├─────────────────────────────────┤
│  Bottom CTA (메뉴 탭 only)       │
│  [주문/예약하기]                   │
└─────────────────────────────────┘
```

- **max-width**: `480px`, 중앙 정렬 (`page-container`)
- **배경**: `bg-surface-0`

## 3. Hero 이미지 캐러셀

- **스타일**: 가로 스크롤 리스트, `gap-4`, `pl-4 pr-4`
- **카드 너비**: `75%` → 다음 이미지가 오른쪽에 살짝 보임
- **비율**: `aspect-[4/3]`, `rounded-2xl`
- **최대 표시**: 12장, 초과 시 "더보기 +N" 카드
- **정렬**: primary 이미지가 항상 첫 번째
- **이미지 없을 때**: 그라데이션 배경 + CakeIcon placeholder

## 6. 메뉴 탭 (`menu`)

### 6-1. 예약 가능 날짜 피커
- 상단 카드, `ClockIcon` + "예약 가능 날짜"
- `<select>`로 14일 이내 날짜 선택 (요일 표시)

### 6-2. 메뉴 리스트 (new: shop_menus)
각 메뉴는 **클릭 가능한 카드** (`<button>`)

```
┌──────────────────────────────────┐
│ [이미지] 메뉴이름           가격 > │
│   설명 (truncate)                 │
│ [📷][📷][📷][📷]  ← 포트폴리오 미리보기│
└──────────────────────────────────┘
```

- **이미지**: `h-12 w-12 rounded-lg`
- **가격**: 사이즈별 가격 범위 또는 단일 가격
- **포트폴리오 미리보기**: `h-14 w-14`, 최대 4장, +N 표시
- **클릭 시**: 메뉴 상세 풀페이지 팝업 오픈 (`openMenuPopup`)

### 6-3. 레거시 메뉴 (shop_menu_prices)
- `shop_menus`가 비어있을 때만 표시
- 사이즈 탭 필터 + 디자인 타입별 가격 리스트

### 6-4. 주문/예약하기 CTA (바이어 only)
- 메뉴 탭에서만 하단 고정
- `sticky bottom-0`, 그라데이션 버튼
- 셀러에서는 **표시 안 함** (대신 편집 버튼)

---

## 7. 메뉴 상세 풀페이지 팝업

> ⚠️ **중요**: `menuPopupStack` 스택 기반으로 뒤로가기 지원

### 7-1. 팝업 헤더
- ← 뒤로 (popMenuPopupView) / 메뉴 이름 or "포트폴리오" / ✕ 닫기
- 스택 깊이 > 1이면 ←, ✕ 둘 다 표시

### 7-2. 메뉴 상세 뷰 (`type: 'menu'`)
```
┌─────────────────────────────┐
│ [메뉴 대표 이미지 16:9]      │
│ 메뉴 이름 (text-lg bold)     │
│ 설명 (pre-wrap)              │
│ ┌──────────────────────────┐│
│ │ 미니 | 15,000 ~ 20,000원  ││
│ │ 1호  | 25,000 ~ 35,000원  ││
│ └──────────────────────────┘│
│ 📷 포트폴리오 (N)            │
│ [사진1] [사진2]              │
│ [사진3] [사진4]              │
└─────────────────────────────┘
```

- 사이즈별 가격: `rounded-lg bg-surface-50 px-3 py-2.5`
- 포트폴리오: 2열 그리드, 그룹 커버 + +N 뱃지
- 포트폴리오 클릭 시: 포트폴리오 상세 뷰로 `pushMenuPopupView`

### 7-3. 포트폴리오 상세 뷰 (`type: 'portfolio'`)
- 그룹 이미지 캐러셀 (가로 스크롤 + dot indicator)
- 설명 텍스트
- 메타 정보: 메뉴명, 사이즈, 가격, 제작일
- 등록 날짜

## 8. 공지 탭 (`notices`)

```

│ [고정] 📌 공지 제목            │
```

- 셀러: 추가로 "공지 관리하기" 편집 버튼

## 9. 사진 탭 (`photos`)

### 9-1. 사진 그리드
- `grid-cols-3 gap-1.5`
- 그룹 사진은 커버 이미지만 표시 + 우하단 `+N` 뱃지
- 포트폴리오 뱃지: 좌상단 `bg-primary-500/80`
- **데이터 소스**:
  - 바이어: `galleryImages + reviewPhotos` 합산
  - 셀러: `galleryImages (is_portfolio 포함)` 별도, `reviewPhotos` 별도

### 9-2. 사진 클릭 동작
- **바이어**: 풀스크린 포토 뷰어 오픈 (그룹 전체 이미지, 이전/다음 내비게이션)
- **셀러**: Link로 `/portfolio/{id}` 이동 + `⋮` 컨텍스트 메뉴 (대표사진 등록/삭제)

### 9-3. 셀러 전용
- "사진올리기" 버튼 → 바텀시트 (일반사진/포트폴리오 선택)
- 셀러 Primary(★ 대표) 뱃지 표시

## 10. 리뷰 탭 (`reviews`)

### 10-1. 별점 요약
```
┌──────────────────────────────┐
│  4.5     ★ 5 ████████ 8     │
│  ★★★★★  ★ 4 ███      2     │
│  12개의 리뷰  ★ 3 ██      1     │
│           ★ 2 █       0     │
│           ★ 1          1     │
└──────────────────────────────┘
```

### 10-2. 필터 & 외부 링크
- `☑ 사진 포함 리뷰만` 체크박스 필터
- 외부 리뷰 링크: 네이버(초록), 카카오(노랑), 구글(파랑) pill 뱃지

### 10-3. 리뷰 카드
```
[아바타] 닉네임      2026.04.20
         ★★★★☆
리뷰 내용 (text-sm)
[리뷰 사진 (클릭 시 뷰어)]
```

- 아바타: `avatar_url` 있으면 이미지, 없으면 닉네임 첫 글자
- 사진: `h-48 w-full object-cover`
- 양쪽 동일

## 11. 정보 탭 (`info`)

### 11-1. 픽업 안내
- 아코디언 UI: 제목 클릭 시 설명 펼침
- 이미지 지원 (`image_urls` 배열)

### 11-2. 영업시간
- 요일별 시간 표시 (월~일)
- 오늘 요일 강조 (`bg-primary-50` + "오늘" 뱃지)
- 휴무일: "휴무" 회색 표시

### 11-3. 위치
- 주소 텍스트
- 네이버 지도 링크 (`naver_map_url` → "네이버 지도에서 보기")
- 없으면: "지도에서 검색하기" (주소 기반 검색)

### 11-4. 판매자 정보
- 아바타 + 닉네임 + 이메일
- 양쪽 동일

## 13. 셀러(내 스토어) 전용 기능

| 기능 | 위치 | 설명 |
|---|---|---|
| 메뉴 편집/추가 | 메뉴 탭 하단 | "메뉴 편집하기" 토글, "메뉴 추가" 풀페이지 모달 |
| 공지 관리 | 공지 탭 하단 | "공지 관리하기" 토글, CRUD |
| 사진 ⋮ 메뉴 | 사진 썸네일 우상단 | [대표사진 등록 / 삭제] 드롭다운 |
| 사진올리기 | 사진 탭 하단 | 바텀시트 (일반사진/포트폴리오) |
| 정보 편집 | 정보 탭 하단 | "정보 편집하기" 토글, 모든 필드 편집 |
| + 플로팅 버튼 | 우하단 | `/seller/portfolio/new` 이동 |
| 하단 탭 바 | 페이지 하단 | 대시보드/주문/스케줄/채팅/내 스토어 |
